Frequently Asked Questions

How do I install HandyWorks?

  1. Install MS-Access on that machine
  2. Start Access, go to the trust center and enable all macros
  3. Download the install helper file Helper and then do the latest upgrade
  4. Drag the HW shortcut for the program from the hww folder to the desktop
  5. Once you start the program, it will look for the patient data file

Can I make copies for my other offices?

Of course not! Each separate office location needs its own (purchased copy). You can run multiple clinics from within one copy of HandyWorks by adding them in the Preferences screen.

How do I network HandyWorks?

  1. Make sure the /hww folder on the main computer is shareable with read/write ability
  2. Or have your data in the Users/Public/Documents folder
  3. Map the \hww folder or the documents folder to a drive letter (we recommend that you call it H:)
  4. When program opens, it will ask where the data is
  5. Point it to hwdata in H:
