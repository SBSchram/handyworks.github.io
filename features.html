HandyWorks Features

HandyWorks has many of the same features found in other management programs. Below is a short list of all the things that HandyWorks can do for you.

Core Features

  • Appointments - How appointments are handled, along with report options
  • Accounting - Significant accounting procedures
  • Billing Wizard - How the complex handling of insurance claims is done
  • Daysheets and Statistics - Summary reports for varied period of time are quite remarkable
  • Support Files - Doctors, attorneys, carriers, referral sources
  • Letters and Paragraphs - Source for writing letters, narratives and SOAP notes

  • Security - How is security implemented
