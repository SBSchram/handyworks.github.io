2/13/2017 Update

2017-02-13 Update Blog

A few gentle fixes are included in the 2/13/2017 update:

  • First: A fix in electronic bills, when there are multiple locations and a master billing clinic, the NPI number for the service location is left blank. The NPI for the billing provider and rendering provider are included.
  • Secondly: The calculation of visit counts for Day Stats, when there were multiple clinics, was not totaling correctly.
  • Lastly: There is new code in the Main Switchboard that will pick up if you are not running the proper version of HandyWorks for your program or for your HWDATA.
