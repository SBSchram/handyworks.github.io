A longer Payer ID - Fall 2022 update

2022-10-07 Update Blog

Wow! 5 characters are not enough. So we expanded the field to 9 in Carriers. Use it if you need it. Otherwise, there is no difference for most of you.

Also, we finally squashed a reporting bug that occurred when closing the program. It was happening when a record in case data was not saved. It was being automatically saved when the program closed, but a subsequent update function wouldn't work. All fixed now. No more error messages if colosing with an unsaved case data record.
